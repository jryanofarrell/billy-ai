**Key decisions:**
- `PLAYWRIGHT_BROWSERS_PATH=0` is the single load-bearing contract: the build script installs Chromium into the playwright package dir with this var set, the spec discovers it there and adds it to PyInstaller `datas`, and the frozen `__main__.py` sets the same var before any Playwright import so the runtime finds the bundled browser. One-folder (not one-file) build keeps Chromium at stable relative paths and avoids per-launch extraction overhead.
